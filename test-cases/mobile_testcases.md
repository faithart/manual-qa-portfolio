- Дата проверки: 2025-12-01

## TC-MBL-01 — Пуш появляется в шторке при закрытом приложении
**Предусловия:** 
1. Установить версию приложения v.2.0 на эмулятор Pixel 3a
2. Открыть приложение
3. Ввести в поле Backend url ... 
4. Через запрос POST /api/v1/courier в Postman создать курьера с логином testcourier, паролем 1234 и именем Наруто
5. Залогиниться в приложении
6. Создать заказ запросом POST /api/v1/orders в Postman с телом (см. в примечании) на дату доставки завтра
**Примечания:**
{
    "firstName": "Иван",
    "lastName": "Петров",
    "address": "ул Ленина 15",
    "metroStation": 4,
    "phone": "+7 999 999 99 99",
    "rentTime": 1,
    "deliveryDate": "2025-11-25",
    "comment": "",
    "color": ["BLACK"]
}
**Шаги:**
1. Перейти на вкладку "Все"
2. Принять заказ 
3. Закрыть приложение
4. Изменить в настройках время и дату устройства на следующий день (26.11.2025, 21:58)
5. Дождаться 21:59, обратить внимание на пуш
**Ожидаемый результат:** В шторке появляется пуш-уведомление от приложения курьера "2 часа до конца заказа"
**Статус:** Passed
