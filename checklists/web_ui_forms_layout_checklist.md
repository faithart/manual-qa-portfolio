- [x] Кнопка "Заказать" находится в правом верхнем углу, представляет собой чёрный прямоугольник со скруглёнными углами и белым текстом.
- [x] Справа от кнопки "Заказать" расположена ссылка "Статус заказа": шрифт черный, нежирный.
- [x] Поле ввода номера заказа — белый прямоугольник с чёрным тонким контуром и закруглёнными углами.

## 3. Логика работы
- [ ] При нажатии на кнопку "Статус заказа" открывается поле ввода номера заказа с плейсхолдером "Номер заказа".  
  ⤷ Дефект: [B3](../bug-reports/B3.md)
- [ ] При вводе номера заказа нужно нажать клавишу Enter, тогда появляется информация о заказе.  
  ⤷ Дефект: [B4](../bug-reports/B4.md)
- [x] При вводе несуществующего номера заказа появляется ошибка "Такого заказа нет. Точно верный номер?".
- [x] Если введён номер существующего заказа, появляется информация, соответствующая заказу: имя, фамилия, адрес, срок аренды, цвет и др.
