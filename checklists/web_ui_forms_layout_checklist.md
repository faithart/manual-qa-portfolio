# Web UI — формы и вёрстка (чек-лист)

## Окружение
- Браузер: Chrome версия 142.0.7444.176 (64-bit)
- Доп.браузер: Яндекс Браузер  версия 25.10.1.1136 (64-bit)
- Разрешения: 1920x1080 (Chrome), 1280×720 (Яндекс Браузер)
- Дата проверки 08.12.2025
- ОС: Windows 11

## 1. Формы
- [ ] Валидация обязательных полей
- [ ] Сообщения об ошибках понятные и не перекрывают элементы
- [ ] Нельзя отправить пустую форму
- [ ] Поля принимают допустимые форматы (email/телефон и т.п.)
- [ ] Негативные данные (пробелы, спецсимволы, слишком длинные значения)

## 2. UX / удобство
- [ ] Понятные подписи/плейсхолдеры
- [ ] Фокус в полях (Tab-порядок)
- [ ] Кнопки доступны и не “прыгают” при ошибках

## 3. Вёрстка и адаптив
- [ ] Нет горизонтального скролла на мобильных разрешениях
- [ ] Текст не обрезается, кнопки не налезают
- [ ] Элементы не “уезжают” при длинных значениях

## 4. Ошибки и состояния
- [ ] Понятная обработка 4xx/5xx (если применимо)
- [ ] Лоадеры/спиннеры не зависают
