# Баг-репорты

<a id="b-1"></a>
## B1 — Ошибка в поле Адрес при вводе 50 символов
**Приоритет:** Стандартный  
**Статус:** Открыт  

### Окружение
Windows 11 Домашняя для одного языка, браузер Google Chrome, версия 142.0.7444.176 (64-bit), тестовый стенд https://f7fe24b0-b049-4604-b474-c78969448453.serverhub.praktikum-services.ru/

### Предусловия
1. Открыть тестовый стенд https://f7fe24b0-b049-4604-b474-c78969448453.serverhub.praktikum-services.ru/
2. Нажать кнопку "Заказать"

### Шаги воспроизведения
1. Ввести в поле Адрес `Улица Лестная дом 25 корпус 7 квартира 10 строение`
2. Снять фокус с поля

### Ожидаемый результат
Рамка поля становится черной, ошибок нет.

### Фактический результат
Красная рамка поля, ошибка: "Введите корректный адрес".
